# Genealogy Photo Documentation — Naming Conventions
## Names

* Last names in ALL CAPS (e.g., John SMITH)
* Maiden names in ALL CAPS inside parentheses (e.g., Mary (JORDAN) Smith)
* Nicknames in "quotes" (e.g., William "Bill" SMITH)
* Always use birth/maiden names, not married names
* Use underlines for unknown names so someone can fill them in later (e.g., _____ SMITH)

## Dates & Places

* Use "Abt" for approximate dates or places (e.g., Abt 1921, Abt Chicago)

## General Tips

These conventions apply to captions, labels, and genealogy software notes
If naming digital files, avoid parentheses and special characters — some systems can't handle them
Consistency is the most important rule — pick a format and stick to it
